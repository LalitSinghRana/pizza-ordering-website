# Pizza ordering website (MERN stack)

## STEPS TO RUN :

### To Complete setup :

- Git clone this repo.
- Create .env file (using .env.example) :
  - insert your
    - mongodb URL,
    - email id and password (for sending emails)
- Install all dependencies :
  - For client :
    - > cd client && npm i
  - For server :
    - > cd server && npm i

### To run :

- To run client :
  - > cd client && npm start
- To run server :
  - > cd server && npm start
